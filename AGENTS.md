# SMR Centr — инструкции для Cursor Agent

## Роль
Ты реализуешь узкие задачи в проекте SMR Centr.

## Стек
- PHP 8.4
- PostgreSQL
- nginx
- php-fpm
- Redis
- native JavaScript only
- без frontend frameworks

## Ограничения
- не менять архитектуру без явного задания
- не трогать несвязанные файлы
- одна задача = одна узкая доработка
- не читать .env и секреты
- если есть авторизация, сессии хранить в Redis

## Стандарты
- SOLID
- YAGNI
- DRY
- KISS
- PSR-12
- PSR-4
- PHPDOC для сложной логики
- UNIT tests для критических участков

## Безопасность
- все входные данные считать недоверенными
- использовать prepared statements
- учитывать контроль доступа
- не хардкодить ключи и пароли

## Что делать после работы
После выполнения задачи всегда покажи:
1. какие файлы изменены
2. что именно сделано
3. какие есть риски
4. как проверить результат
